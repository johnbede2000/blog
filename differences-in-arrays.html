Differences in Arrays

21/10/20

Challenge

"Compare two arrays and return a new array with any items only found in one of the two given arrays, but not both." - freeCodeCamp (Intermediate Algorithm Scripting)

Description of Solution

  1. First we'll create an empty array to return the selected elements in: let returnedArr = [];.

  2. Next we'll loop through one of the arrays, and compare each element with the other array. There are different ways to loop; the forEach() method is one way. So arr1.forEach(insert function here)

  3. The function we'll perform on each element from arr1 is: '1) is it the same as any of the elements in arr2? 2) If it's not, then we'll push it to our returned array.'

    1. To compare with all of the elements in arr2, we could use the some() method. So arr2.some(arr2Elem => arr1Elem === arr2Elem)
    2. If this returns false, then we can move onto pushing the element. So we need to wrap the above in an if statement. If the above returns true we're not going to do anything. So it's easier to only write the code for false, so let's reverse the logic to arr1Elem !== arr2Elem - this would need to be the case for all elements not just some. So we'll use the every() method instead. arr2every(arr2Elem => arr1Elem !== arr2Elem)
    3. and finally the code to execute when this is the case will be returnedArr.push(arr1[elem])

  4. We'll now need to do steps 2 and 3 the other way round as well! Comparing array2 with array1.

  5. Finally we need to return returnedArr.

Solution

function diffInArrays(arr1, arr2) {
    let returnedArr = [];
    
    arr1.forEach(arr1Elem => {
        if (arr2.every(arr2elem => arr1Elem !== arr2elem)) {
            returnedArr.push(arr1Elem);
        };
    });
    
    arr2.forEach(arr2Elem => {
        if (arr1.every(arr1Elem => arr2Elem !== arr1Elem)) {
            returnedArr.push(arr2Elem);
        };
    });
    
    return returnedArr;
}
