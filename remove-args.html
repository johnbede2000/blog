Remove Arguments from Array

Challenge

You will be provided with an initial array (the first argument in the destroyer function), followed by one or more arguments. Remove all elements from the initial array that are of the same value as these arguments.

Description of my solution

  1. We don't know how many arguments will be provided, so we'll use the rest parameter ...args. This will provide an array that we can use inside the function.
  2. We'll iterate through every element in the initial array to see if it's listed in the args array. We could use forEach() with an if statement to push elements that pass to a new array (which the function returns).. Or instead we can just use the filter() method which will do the same thing with less lines. (The filter method creates a new array and doesn't modify the original.)
  3. The way that we'll see if the element is present as one of the arguments: if it is present, then the indexOf method will return its index. If the element is not present, the indexOf method will return -1. So to write this last sentence the other way round, if the element does have an index of -1, we know it's not present. And therefore safe to keep in our filtered array.

My Solution

function removeArgs(arr, ...args) {
    return arr.filter(elem => (args.indexOf(elem) == -1));
}
